### 最终查询流程

#### 主查询（4 张表关联）
```sql
SELECT DISTINCT
    ct.id as courseTableId,
    ct.course_id as courseId,
    ct.course_name as courseName,
    c.course_code as courseCode,         -- 从 tb_course 表获取
    ct.college_name as collegeName,      -- 冗余字段
    cc.id as courseCategoryId,
    cc.course_category_name as courseCategoryName,
    g.id as groupId,
    g.group_name as groupName,
    g.group_no as groupNo,
    g.sort as groupSort,
    g.source as groupSource
FROM tb_course_table ct
INNER JOIN tb_course c ON ct.course_id = c.id         -- 新增 JOIN
INNER JOIN tb_group g ON ct.group_id = g.id
INNER JOIN tb_group_member gm ON g.id = gm.group_id
LEFT JOIN tb_course_category cc ON ct.course_category_id = cc.id
```

#### 班级人数查询
```sql
SELECT group_id, COUNT(*) as member_count
FROM tb_group_member
WHERE group_id IN (?) AND group_member_status = 0
GROUP BY group_id
```

#### 教师信息查询
```sql
SELECT ct.id as courseTableId, t.teacher_id, t.teacher_name
FROM tb_course_table ct
INNER JOIN tb_course_table_detail ctd ON ct.id = ctd.course_table_id
INNER JOIN tb_course_table_detail_teacher t ON ctd.id = t.course_table_detail_id
WHERE ct.id IN (?)
ORDER BY t.show_order
```
